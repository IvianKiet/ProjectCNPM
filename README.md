=====The system supports multiple roles:

\- Guest

\- Restaurant Owner

\- Staff

\- Kitchen

\- Admin

=====Main features:

\- Authentication and authorization

\- Order management

\- Restaurant and table management

\- Revenue management

\- AI chatbot integration

\- Database-driven backend architecture

=====Technologies Used

\- Backend: Python

\- Framework: FastAPI (or similar Python framework)

\- Database: SQL-based database

\- AI: Python AI chatbot

\- Containerization: Docker, Docker Compose

\- Frontend: HTML, CSS, JavaScript (developed separately)

=====Backend Requirements

All backend dependencies are defined in:

backend/requirements.txt

=====Install dependencies

Make sure Python 3.10 or newer is installed, then run:

pip install -r backend/requirements.txt

If you use Docker, dependencies will be installed automatically.
